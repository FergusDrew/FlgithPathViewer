    • The Basics

    • Time and Date selection

      To alter the time and date filters first enter your start date from the missions you want to view. From then you can then choose a start date as the end time of the missions you wish to view, if not then all mission flights after this data will be shown on the map.

    • Time Analytics

      This section will display to you all the temporal data. If you have not selected any dates then all the data loaded will be displayed and will be calculated to give you usable data. But if you select dates and times you want to filter through then the data from those times will be shown.

    • Area coverage analytics

      The same as the time analytics this will show all the area coverage of the loaded data should you not select your prefferences in time but if they are selected then it will show you them for the current flights and area that was selected.

    • Tagging

      The tagging system can be used for separating the different flight data you want to display. First a tag must be created and named. Then you must click on the select tags to choose which flights you wish to add to the tag. Then you can confirm your choices to see only those selected in a different colour. You can also edit the tag settings and delete said tag.
